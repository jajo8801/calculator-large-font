About this app

    Instructions for using the app

  • Use WASD for up, left, down, and right selection movements.
  • Also use , (COMMA) for left and . (PERIOD)for right movements.
  • Use the space bar to select the key for both interfaces.


    • Test

  • Time the tasks using a stopwatch of your choice and type the following equaitons without errors.
  • At the end of the exercise repeat two more times then average the times for each trial
  • Each equation ends with "=C" meaning you should select the "=" button and the "C" button on the calculator after each equation
